-------------------------------
🌐 Demo en vivo:
https://pokedex-2024-1.netlify.app/
----------------
📌 Descripción del Proyecto

- Desarrollé una aplicación web Pokédex que permite visualizar, buscar y explorar información detallada de diferentes Pokémon mediante una interfaz moderna e intuitiva.

- La aplicación permite a los usuarios consultar el nombre, tipo e imagen de cada Pokémon, facilitando la exploración de la base de datos de forma rápida e interactiva.

- Este proyecto forma parte de mi portafolio como Desarrollador Web y demuestra mis habilidades en consumo de APIs, desarrollo frontend y diseño responsive.

----------------
🚀 Características Principales

✔ Visualización de lista de Pokémon

✔ Búsqueda por nombre

✔ Visualización de imagen y tipo

✔ Interfaz intuitiva y amigable

✔ Diseño responsive

✔ Navegación fluida

✔ Despliegue en Netlify

El objetivo fue desarrollar una aplicación web interactiva que consuma datos dinámicos desde una API y los muestre en una interfaz amigable, aplicando buenas prácticas de desarrollo frontend.
